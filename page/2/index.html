Js函数

在Javascript中定义一个函数，有两种写法：

  • function foo() { }
  • var foo = function () { } ， 建议使用的写法

两种写法完全等价。但是在解析的时候，前一种写法会被解析器自动提升到代码的头部，因此违背了函数应该先定义后使用的要求，所以建议定义函数时，全部采用后一种写法。

案例1： 以下是个人对js这种语法的一种理解。下面可以拆分为几段

  1. (function($){})(jQuery);这个会比较熟悉一点就是声明一个匿名函数然后调用，jQuery是传的参数，是Jquery框架的一个对象。
  2. $.extend({});表示继承$，而这里的$就是传进来的jQuery，然后实现一些自己的方法，方便直接通过$.function的形式调用
  3. {what: function(string){}} 在对象体中可以这样定义函数。var a = {what: function(){}}通过a.what(); 调用
