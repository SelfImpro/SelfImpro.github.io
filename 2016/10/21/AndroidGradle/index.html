Build Tasks

在Gradle架构中，每一个任务都会生成一个Task，可以直接运行。

基础任务

由android插件提供的功能，平时看到的apply plugin: 'com.android.application'这个就是引入android插件。

  • assemble

    The task to assemble the output(s) of the project.
  • check

    he task to run all the checks.
  • build

    This task does both assemble and check.
  • clean

    This task cleans the output of the project.
