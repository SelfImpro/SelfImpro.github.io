Web前端学习第一天

发表于 2016-09-20   |     |   阅读次数

最近由于公司需要开发Web APP，所以正式从这里开始接触H5开发。从这里开始记录一些学习笔记，供后面复习。

IDE

开发的IDE我选择了Sublime 和 WebStorm

Sublime 插件安装

对于Sublime需要安装一些插件辅助开发

  1. command + shift + p打开安装命令行
  2. 输入 install , 选择Install Package这个是加载仓库去了，等待几分钟会继续弹一个窗口出来
  3. 输入要安装的插件名称

对于H5开发安装下面几个插件就够了

  • Emmet提供一些快速生成标签的功能
  • HTML-CSS-JS Prettify。 （这个插件有点坑就是项目的路径里面不能含有中文，否则会报Node.js was not found in the default path. Please specify the location的错误）
  • SideBarEnhancements增强右键功能Sublime Text 2版本下载
  • TrailingSpaces 下载点这里
  • CTags

CTags 安装的坑

CTags够坑，默认安装后运行报错

解决方式如下

  1. 手动下载安装包>>>点击下载<<<

  2. 执行以下命令

mac下载默认安装到/usr/local/bin下面。接下来就配置sublime，让它的ctags指向这个路径就可以了。

  1. 打开菜单在Preferences菜单中打开Package settings->ctags->settings-user和settings-default，把default中的配置全部复制到user中，将配置中command的值设置为/usr/local/bin/ctags。

mac sublime 手动安装插件的坑

手动安装插件从git仓库将插件放到Packages目录下面，后面有一点要做的就是记得把手动安装的插件在Package Control配置文件中注册一下，不然下一次进来就没有了，被挪到backup目录下面了。

行内元素和块级元素

下面是自己理解的关于这两种类型的差异

类别 布局显示差异 使用场景 标签
行内元素 1.总是另起一行开始；
2.高度，行高以及顶、底边距都可控制；
3.宽度缺省是它所在容器的100%，除非设定一个宽度		 span
块级元素 1.总是另起一行开始
2.高度，行高以及顶、底边距不可改变；
3.宽度就是它所容纳的文字或图片的宽度，不可改变		 一般用于构造页面结构，一般作为容器使用 div p table
